Regular Expressions Tutorial

Regular expressions are patterns used to match character combinations in strings.

Each input box has the regular experssion to validate the input listed above with a short explanation.

Email:

Regular Expression Pattern: /^[^@\s]+@[^@\s]+\.[^@\s]+$/i


Phone:

Regular Expression Pattern: /^[0-9]{10,11}$/g

The ^ matches the beginning of the input. The [0-9] matches a character in the range 0 to 9. The {10,11} requires the set to be between 10 and 11 characters. The $ matches the end of the string.


Password:

Regular Expression Pattern: /^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)(?=.*[$@$!%*?&])[A-Za-z\d$@$!%*?&]{8,}/g
